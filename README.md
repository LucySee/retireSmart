# RetireSmart – Financial Independence Simulator

A free, privacy-first multi-asset retirement simulator. No accounts, no cloud storage, no ads. All data stays in your browser.

## Features

- **Multi-asset panels** – EPF, Fixed Deposit, Savings, Stocks, ETF (fully editable)
- **Per-asset settings** – different return rates, contribution frequencies (monthly/quarterly/yearly)
- **Retirement age detection** – auto-detects the year your portfolio hits your FI number
- **Portfolio projection chart** – combined or per-asset breakdown
- **Inflation toggle** – nominal vs real (inflation-adjusted) view
- **Scenario comparison** – test what-if scenarios vs your base plan
- **Privacy mode** – hide all values with one click
- **Quick reset** – wipe all data instantly

## 🚀 Deploy to GitHub Pages (Free hosting)

### Step 1 – Create a GitHub repository

1. Go to [github.com](https://github.com) and log in
2. Click **New repository**
3. Name it: `retiresmart` (or any name you like)
4. Set it to **Public**
5. Click **Create repository**

### Step 2 – Upload the files

**Option A – Upload via browser (easiest):**
1. On your new repo page, click **uploading an existing file**
2. Drag and drop all files in this folder (`index.html`, `manifest.json`, `README.md`)
3. Click **Commit changes**

**Option B – Via Git command line:**
```bash
cd retirement-simulator
git init
git add .
git commit -m "Initial release – RetireSmart"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/retiresmart.git
git push -u origin main
```

### Step 3 – Enable GitHub Pages

1. Go to your repo → **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **main** branch and `/ (root)` folder
3. Click **Save**
4. Wait ~60 seconds, then your app is live at:

```
https://YOUR_USERNAME.github.io/retiresmart/
```

Share this link with anyone — it works on mobile and desktop!

## 📱 Mobile Installation (No App Store needed)

Users can install RetireSmart like a native app:

**Android (Chrome):**
1. Open the GitHub Pages link in Chrome
2. Tap the ⋮ menu → **Add to Home Screen**
3. App icon appears on home screen

**iPhone (Safari):**
1. Open the link in Safari
2. Tap the Share button → **Add to Home Screen**
